# Desempenho Acadêmico com base em Dados

Este projeto apresenta uma análise de dados educacionais com o objetivo de compreender fatores que influenciam o desempenho acadêmico dos estudantes.

## Objetivo

Investigar a evolução das notas e identificar variáveis acadêmicas e comportamentais associadas ao desempenho final.

## Tecnologias utilizadas

- Python
- Jupyter Notebook
- Pandas
- Matplotlib / Seaborn
- Power BI

## Etapas do Projeto

1. Análise exploratória de dados no Jupyter Notebook
2. Investigação de relações entre variáveis acadêmicas
3. Construção de indicadores e visualizações
4. Desenvolvimento de dashboard interativo no Power BI

## Principais análises

- Evolução das notas (G1, G2 e nota final)
- Distribuição do desempenho acadêmico
- Relação entre faltas e desempenho
- Impacto do tempo de estudo
- Histórico de reprovações

## Dashboard

O dashboard apresenta duas visões principais:

- Panorama geral do desempenho acadêmico
- Fatores acadêmicos e comportamentais associados à nota final
